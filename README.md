🌿 Why IQRA?
IQRA (اقْرَأْ) was the first word revealed by Allah ﷻ to Prophet Muhammad ﷺ through Angel Jibril.
The project is inspired by that first revelation and aims to make Qur'an reading, listening, structured learning and reflection accessible through a beautiful, respectful and modern interface.
> **Mission:** Build one of the most thoughtful open-source Qur'an learning platforms.

🚀 Installation
```bash
git clone https://github.com/arzoochelsea/iqra.git
cd iqra
npm install
npm run dev
```
Production:
```bash
npm run build
npm start
```
